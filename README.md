# Firdha Dewi Oktaviani — Portfolio

Website portofolio profesional untuk **Firdha Dewi Oktaviani**, seorang *Behavior Therapist* & *Social Worker*. Proyek ini merupakan hasil migrasi dari HTML statis ke arsitektur modern menggunakan React dan Vite.

## 🚀 Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite 8](https://vite.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Language:** TypeScript
- **Icons:** Emoji-based & Custom CSS

## ✨ Fitur Utama

- **Migrasi Modern:** Performa tinggi dengan Vite HMR (Hot Module Replacement).
- **Tailwind 4 Ready:** Menggunakan engine terbaru Tailwind CSS v4 untuk styling yang lebih efisien.
- **Animasi Smooth:** Implementasi *Intersection Observer* untuk efek *fade-up* saat scroll.
- **Hover Effects:** Animasi interaktif pada bagian pengalaman (*Experience*) dan layanan.
- **Responsif:** Desain yang dioptimalkan untuk berbagai ukuran perangkat (Desktop, Tablet, Mobile).
- **Profil Visual:** Integrasi foto profil dengan efek overlay di bagian Hero.

## 🛠️ Pengembangan

### Instalasi Dependensi
```bash
npm install
```

### Menjalankan Mode Pengembangan
```bash
npm run dev
```

### Build untuk Produksi
```bash
npm run build
```

## 📁 Struktur Proyek Utama

- `src/App.tsx`: Komponen utama yang berisi seluruh konten dan logika portofolio.
- `src/index.css`: Definisi tema Tailwind 4, variabel CSS, dan gaya kustom.
- `public/profile.webp`: Foto profil utama yang digunakan di bagian Hero.
- `index.html`: Entry point HTML dengan konfigurasi Google Fonts.
